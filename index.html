予定を追加

スコープ calendar.events が必要です。OAuth 設定後に有効になります。

OAuth 設定

Google Cloud で発行した Client ID と API Key を入力します。 値はブラウザの localStorage に保存され、空の config.json より優先されます。

Authorized JavaScript origins に https://toyfer.github.io と localhost を追加してください。Client Secret は不要です。
